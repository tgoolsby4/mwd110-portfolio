The Fibonacci Sequence



The Fibonacci Sequence is a classic mathematics problems in which a collection of fibonacci numbers is given. A specific fibonacci number denoted Fn is the sum of the two previous fibonacci numbers.


There are some preliminary base cases for the Fibonacci Sequence. The first and second fibonacci numbers are both 1. To give this in terms of fibonacci numbers; F1 = 1, F2 = 1. Therefore, in order to caluculate F3, we would have to take the sum of the two previous fibonacci numbers, so F3 = F1 + F2. If a value of 0 is provided, the resulting Fibonacci Sequence will simply have a value of 0 since there are no previous values to add together.
